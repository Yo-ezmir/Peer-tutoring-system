# Peer-tutoring-system

This C++ console application allows students and tutors to register, schedule, and manage tutoring sessions.

## Features

- User registration for students and tutors
- Students can schedule sessions with available tutors
- Tutors can approve or reject session requests
- Data is saved in simple text files
